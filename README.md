# Hangman Game

A simple text-based Hangman game developed using Python. The program randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player has a maximum of 6 incorrect attempts to guess the word correctly. This project demonstrates the use of Python concepts such as loops, conditional statements, lists, strings, and the random module.

## Features
- Random word selection
- Letter-by-letter guessing
- Maximum 6 incorrect attempts
- Console-based gameplay
- Simple and user-friendly interface

## Technologies Used
- Python 3
